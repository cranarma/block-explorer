## Block Explorer

This project contains a block explorer via a REST API built on ExpressJS
and a VueJS-based frontend.

NodeJS version: 10.15.1
NPM version: 6.4.1

App tasks:
  `npm run build-app`

Server tasks:
  `npm run build-server`
  `npm run build-server:watch`
  `npm run start-server`
  `npm run dev-server`
